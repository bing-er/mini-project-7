# Data Instructions

The dataset is NOT included in this repository due to file size constraints.

## Dataset: BCCD (Blood Cell Count and Detection)
- Source: Roboflow Universe
- URL: https://universe.roboflow.com/roboflow-100/bccd
- Classes: RBC, WBC, Platelet
- Size: ~360 images with bounding box annotations
- Format: YOLOv8

## How to Download

1. Sign up for a free account at https://app.roboflow.com
2. Go to Settings → API Keys and copy your API key
3. Run the following in the notebook (Section 2):

    from roboflow import Roboflow
    rf = Roboflow(api_key="YOUR_API_KEY")
    project = rf.workspace("roboflow-100").project("bccd")
    version = project.version(1)
    dataset = version.download("yolov8")

4. Set DATASET_DIR = dataset.location in the notebook.
